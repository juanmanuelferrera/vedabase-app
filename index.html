🔒 macOS Gatekeeper Notice

Since this app is not signed with an Apple Developer certificate, macOS Gatekeeper will block it on first launch. This is normal for unsigned apps.

Installation steps:

  1. Unzip the downloaded Vedabase.app.zip file
  2. Move Vedabase.app to your Applications folder

Method 1: System Settings (older macOS versions)

  1. Try to open the app (it will be blocked)
  2. Open System Settings → Privacy & Security
  3. Scroll down and click "Open Anyway"
  4. Confirm by clicking "Open"

Method 2: Terminal (macOS Tahoe and newer)

  1. Open Terminal (press ⌘+Space, type "Terminal", press Enter)
  2. Run this command: xattr -cr /Applications/Vedabase.app
  3. Open the app normally

You only need to do this once. After the first launch, you can open it normally.
